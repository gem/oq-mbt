Welcome to Model Building Toolkit's documentation!
==================================================

The OpenQuake Model Building Toolkit (OQMBT) is a package for building probabilistic seismic hazard source models. This package is created on top of and it fully integrated with the OpenQuake Hazard Modeler's Toolkit (HMTK).
